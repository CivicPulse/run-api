    **alembic/env.py changes:**
    Add environment variable override near the top, after `config = context.config` and before `target_metadata`:
    ```python
    import os
    # Override alembic.ini URL with env var if set (for containerized environments)
    db_url = os.environ.get("DATABASE_URL_SYNC")
    if db_url:
        config.set_main_option("sqlalchemy.url", db_url)
    ```
    This ensures both offline mode (`config.get_main_option`) and online mode (`config.get_section` which reads the same ini option) use the environment-provided URL when running inside a container (K8s init container pattern).
