  - If `attempt_count === 0`: "First visit"
  - Use ordinal helper: 1st, 2nd, 3rd, 4th, etc. The current visit is attempt_count + 1.

Active voter state (`isActive && !recordedOutcome`):
- Full opacity, OutcomeGrid rendered inline below voter info
- `onOutcomeSelect` passed through to OutcomeGrid's `onSelect`

Completed voter state (`recordedOutcome` is set):
- Container has `opacity-60`
- Instead of OutcomeGrid, show a row: `<Check className="h-4 w-4 text-green-600" />` + outcome badge showing `OUTCOME_LABELS[recordedOutcome]`

Skipped voter state (`entry.status === "skipped"`):
- Container has `opacity-40`
- Show "Skipped" text in `text-xs text-muted-foreground`
