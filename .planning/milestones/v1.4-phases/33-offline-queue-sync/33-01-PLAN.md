    Define QueueItem interface:
    ```typescript
    interface QueueItem {
      id: string                          // crypto.randomUUID()

      payload: DoorKnockCreate | RecordCallPayload
      campaignId: string
      resourceId: string                  // walkListId or sessionId

    }
    ```

    Define OfflineQueueState interface with:
    - items: QueueItem[]
    - isSyncing: boolean
    - push: (item: Omit<QueueItem, "id" | "createdAt" | "retryCount">) => void
    - remove: (id: string) => void
    - incrementRetry: (id: string) => void
    - setSyncing: (syncing: boolean) => void
    - clear: () => void

    Create store with `create<OfflineQueueState>()(persist(...))`
    - push: spread item, add id via crypto.randomUUID(), createdAt via Date.now(), retryCount: 0
    - remove: filter items by id
    - incrementRetry: map items, increment retryCount for matching id
    - setSyncing: set isSyncing
    - clear: set items: [], isSyncing: false

    Persist config:
    - name: "offline-queue"
    - storage: createJSONStorage(() => localStorage)
    - partialize: (state) => ({ items: state.items }) -- do NOT persist isSyncing

    Export QueueItem type and useOfflineQueueStore.

    - getSnapshot: () => navigator.onLine
    - getServerSnapshot: () => true (SSR fallback)
