# 82-01 Plan

## Goal

Close the remaining input-validation edge cases from the shakedown with safe, explicit request contracts.

## Tasks

1. Normalize the voter pagination contract so malformed cursors fail safely and `page_size` is either supported explicitly or rejected explicitly with validation.

3. Enforce remaining geometry and phone-bank validation gaps that still allow invalid-but-persisted data, specifically WGS84 polygon coordinates and negative call durations.
