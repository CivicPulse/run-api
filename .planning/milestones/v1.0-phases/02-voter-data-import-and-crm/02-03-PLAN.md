1. **app/services/voter.py** — VoterService class:

   **build_voter_query(filters: VoterFilter) -> Select** — Core composable query builder:
   - Start with `select(Voter)`
   - For each non-None filter field, append condition:
     - `party` / `parties`: exact match / IN clause
     - `precinct`, `city`, `state`, `zip_code`, `county`, `congressional_district`: exact match
     - `age_min` / `age_max`: Voter.age >= / <=
     - `gender`: exact match
     - `voted_in`: `Voter.voting_history.contains([election])` for each election (AND all)
     - `not_voted_in`: `~Voter.voting_history.contains([election])` for each (AND all)
     - `tags`: subquery — voter must have ALL tags (join voter_tag_members -> voter_tags WHERE name IN tags, GROUP BY voter_id HAVING COUNT = len(tags))
     - `tags_any`: subquery — voter must have ANY tag (EXISTS subquery)
     - `registered_after` / `registered_before`: date comparison on registration_date
     - `search`: ILIKE on `first_name || ' ' || last_name` for partial name search
   - Combine conditions with `and_()` or `or_()` based on `filters.logic`
   - Return the query (caller adds pagination)

   **search_voters(session, filters, cursor, limit) -> PaginatedResponse[VoterResponse]** — Execute query with cursor pagination (order by created_at, id). Use established cursor pattern from Phase 1.

   **get_voter(session, voter_id) -> Voter** — Single voter by ID.

   **create_voter(session, data, user_id) -> Voter** — Manual voter creation.

   **update_voter(session, voter_id, data) -> Voter** — Update voter fields.

   **Tag operations** (on VoterService or separate methods):
   - `create_tag(session, name) -> VoterTag`
   - `list_tags(session) -> list[VoterTag]`
   - `add_tag_to_voter(session, voter_id, tag_id)` — Insert into voter_tag_members
   - `remove_tag_from_voter(session, voter_id, tag_id)` — Delete from voter_tag_members
   - `get_voter_tags(session, voter_id) -> list[VoterTag]`

1. **app/api/v1/voters.py** — Voter endpoints:

   **POST /api/v1/campaigns/{campaign_id}/voters/search** — Search/filter voters
   - Accepts VoterFilter JSON body
   - Query params: cursor (optional), limit (default 50, max 200)
   - Calls VoterService.search_voters()
   - Returns PaginatedResponse[VoterResponse]
   - Requires: `require_role("volunteer")` (all campaign members can search)

   **GET /api/v1/campaigns/{campaign_id}/voters/{voter_id}** — Get single voter
   - Returns VoterResponse with full details
   - Requires: `require_role("volunteer")`

   **POST /api/v1/campaigns/{campaign_id}/voters** — Create voter manually
   - Accepts VoterCreateRequest
   - Returns VoterResponse
   - Requires: `require_role("manager")`

   **PATCH /api/v1/campaigns/{campaign_id}/voters/{voter_id}** — Update voter
   - Accepts VoterUpdateRequest (partial update)
   - Returns VoterResponse
   - Requires: `require_role("manager")`

2. **app/api/v1/voter_tags.py** — Tag endpoints:

   **POST /api/v1/campaigns/{campaign_id}/tags** — Create tag
   - Accepts: `{ name: str }`
   - Returns tag object
   - Requires: `require_role("volunteer")`

   **GET /api/v1/campaigns/{campaign_id}/tags** — List all campaign tags
   - Returns list of tags
   - Requires: `require_role("volunteer")`

   **POST /api/v1/campaigns/{campaign_id}/voters/{voter_id}/tags** — Add tag to voter
   - Accepts: `{ tag_id: UUID }`
   - Returns 204
   - Requires: `require_role("volunteer")`

   **DELETE /api/v1/campaigns/{campaign_id}/voters/{voter_id}/tags/{tag_id}** — Remove tag
   - Returns 204
   - Requires: `require_role("volunteer")`

   **GET /api/v1/campaigns/{campaign_id}/voters/{voter_id}/tags** — Get voter's tags
   - Returns list of tags
   - Requires: `require_role("volunteer")`

   **POST /api/v1/campaigns/{campaign_id}/lists** — Create list
   - Accepts VoterListCreate (name, description, list_type, filter_query for dynamic)
   - Returns VoterListResponse
   - Requires: `require_role("manager")`

   **GET /api/v1/campaigns/{campaign_id}/lists** — List all lists
   - Returns PaginatedResponse[VoterListResponse]
   - Requires: `require_role("volunteer")`

   **GET /api/v1/campaigns/{campaign_id}/lists/{list_id}** — Get list details
   - Requires: `require_role("volunteer")`

   **PATCH /api/v1/campaigns/{campaign_id}/lists/{list_id}** — Update list
   - Requires: `require_role("manager")`

   **DELETE /api/v1/campaigns/{campaign_id}/lists/{list_id}** — Delete list
   - Requires: `require_role("manager")`

   **GET /api/v1/campaigns/{campaign_id}/lists/{list_id}/voters** — Get list voters
