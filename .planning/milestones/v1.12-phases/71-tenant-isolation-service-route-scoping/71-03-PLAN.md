**Refactor private helper `_get_question`:**

Change its signature to accept `campaign_id` and JOIN against SurveyScript to enforce script→campaign ownership:
```python
async def _get_question(self, db, script_id, question_id, campaign_id):
    result = await db.scalar(
        select(SurveyQuestion)
        .join(SurveyScript, SurveyScript.id == SurveyQuestion.script_id)
        .where(
            SurveyQuestion.id == question_id,
            SurveyQuestion.script_id == script_id,
            SurveyScript.campaign_id == campaign_id,
        )
    )
    if result is None:
        raise ValueError(f"Question {question_id} not found")
    return result
```
