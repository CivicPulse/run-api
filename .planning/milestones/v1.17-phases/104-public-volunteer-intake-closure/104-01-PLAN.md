# Plan 104-01: Anonymous Public Volunteer Intake

## Goal

Allow valid signup links to accept anonymous volunteer applications without breaking the authenticated prefill path or the approval gate.

## Tasks

- Make public volunteer-application status and submission endpoints accept optional authentication.
- Update volunteer-application intake rules so existing members stay reviewable and rejected applicants can reapply safely.
- Update the public signup route to show the application form to anonymous visitors while keeping authenticated prefill/status behavior.
- Add backend and frontend regression coverage for anonymous submission and the review-safe intake behavior.
