# Plan 102-01: Public Volunteer Application Intake

## Goal

Turn valid signup links into approval-gated volunteer applications without granting campaign access at submission time.

## Tasks

- Add the volunteer application persistence model, schemas, service, API routes, and migration.
- Add authenticated public submission and current-application status endpoints on the signup-link path.
- Replace the signup-link placeholder page with a real application form and status handling for existing users.
- Add focused backend tests for dedupe and submission endpoints.
