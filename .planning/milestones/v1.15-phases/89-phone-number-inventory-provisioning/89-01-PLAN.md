   def get_twilio_client(self, org: object):
       """Return an authenticated Twilio REST client for the given org."""
       from twilio.rest import Client as TwilioClient
       creds = self.credentials_for_org(org)
       if creds is None:

3. Create `app/services/org_phone_number.py` with `OrgPhoneNumberService`:
   - `__init__(self)` stores a `TwilioConfigService` instance
   - `async def register_number(self, db, org, phone_number: str) -> OrgPhoneNumber`:
     a. Check for existing registration (org_id + phone_number); raise HTTPException 409 if duplicate
     b. Get Twilio client via `self._twilio.get_twilio_client(org)`
     c. Call `await asyncio.to_thread(client.incoming_phone_numbers.list, phone_number=phone_number, limit=1)`
     d. Wrap in try/except TwilioRestException: status 401/403 -> HTTP 502 "Twilio credential error"; other -> HTTP 502 "Twilio API error"
     e. If empty list -> HTTP 404 "Phone number not found in your Twilio account"
     f. Extract capabilities dict, friendly_name, sid from result
     g. Determine phone_type: check if twilio_num has a `.type` or `.address_requirements` attr; if not, default to "unknown"
     h. Create OrgPhoneNumber row, add to session, commit, return
   - `async def list_numbers(self, db, org_id: uuid.UUID) -> list[OrgPhoneNumber]`:
     Select all where org_id matches, order by created_at
   - `async def delete_number(self, db, org, number_id: uuid.UUID) -> None`:
     a. Fetch the number row (filter by org_id for isolation); 404 if not found
     b. If org.default_voice_number_id == number_id, set it to None
     c. If org.default_sms_number_id == number_id, set it to None
     d. Delete the row, commit
   - `async def sync_number(self, db, org, number_id: uuid.UUID) -> OrgPhoneNumber`:
     a. Fetch number row (org-scoped); 404 if not found
     b. Get Twilio client, call `await asyncio.to_thread(client.incoming_phone_numbers(number.twilio_sid).fetch)`
     c. Update capabilities, friendly_name, capabilities_synced_at = utcnow()
     d. Commit, return updated number
   - `async def set_default(self, db, org, number_id: uuid.UUID, capability: str) -> None`:
     a. Fetch number row (org-scoped); 404 if not found
     b. Validate the number has the requested capability (voice_capable or sms_capable); 400 if not
     c. If capability == "voice": set org.default_voice_number_id = number_id
     d. If capability == "sms": set org.default_sms_number_id = number_id
     e. Commit
   - `def enrich_response(self, number: OrgPhoneNumber, org) -> OrgPhoneNumberResponse`:
     Build response with is_default_voice and is_default_sms computed from org FK values

<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| client -> API | Untrusted input from browser; phone_number field must be validated |
| API -> Twilio | Outbound call using decrypted org credentials; errors must not leak secrets |
| org -> org | Phone number data must be isolated per org_id; no cross-org access |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-89-01 | Information Disclosure | OrgPhoneNumberService queries | mitigate | All queries filter by org_id derived from JWT via require_org_role; no cross-org access possible. Service methods require org parameter resolved from authenticated user context. |
| T-89-02 | Information Disclosure | TwilioConfigService.get_twilio_client | mitigate | Decrypted auth_token lives only in memory within TwilioClient constructor; never logged or returned in responses. TwilioRestException messages are sanitized before returning to client. |
| T-89-03 | Tampering | RegisterPhoneNumberRequest.phone_number | mitigate | Pydantic schema validates E.164 format (pattern: ^\+[1-9]\d{1,14}$) before any DB or Twilio API call. Twilio API lookup provides server-side validation that number exists in the org's account. |
| T-89-04 | Denial of Service | Twilio API calls in register/sync | mitigate | Mutation endpoints rate-limited at 120/minute via existing slowapi limiter. Twilio API calls wrapped in asyncio.to_thread to prevent event loop blocking. |
</threat_model>
