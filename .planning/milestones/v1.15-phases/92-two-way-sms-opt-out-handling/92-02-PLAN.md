<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Campaign send routes | Authenticated staff actions that must stay within campaign membership |
| Twilio REST send integration | External provider call that must not execute for blocked recipients |
| Bulk queue | Background execution must not bypass the same compliance rules as single send |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-92-03 | Elevation of Privilege | campaign send routes | mitigate | authenticated campaign role checks on every route |
| T-92-04 | Tampering | bulk send worker | mitigate | worker re-runs eligibility checks before every provider send |
| T-92-07 | Information Disclosure | inbox query routes | mitigate | campaign-scoped queries only, no org-wide conversation joins |
</threat_model>
