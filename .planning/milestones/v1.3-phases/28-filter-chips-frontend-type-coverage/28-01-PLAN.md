       - demographics: "bg-blue-100 text-blue-800 dark:bg-blue-900 dark:text-blue-200"

       - Uses formatMultiSelectChip for ethnicities, spoken_languages, military_statuses

       - Registration chips keep existing labels: "City:", "State:", "Zip:"
