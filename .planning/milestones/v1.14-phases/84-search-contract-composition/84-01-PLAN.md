# 84-01 Plan

## Goal

Define a stable backend lookup contract for free-text search without introducing Phase 85/86 search infrastructure early.

## Tasks

1. Normalize free-text lookup input in the voter search service so blank or whitespace-only terms do not change browse behavior.
2. Add a stable default lookup ordering for current name-based search results when no explicit table sort is active.
3. Extend cursor handling so the default lookup ordering can page deterministically across tied results without duplicates or disappearing voters.
