    const record: Record<string, string> = {}

Key implementation details per Pitfall 3:
- Different filter dimensions use different UI patterns:
  - **Checkbox groups** (party, ethnicity, spoken_language, military_status): `page.getByLabel('DEM').check()`
  - **Text inputs** (gender, city, state, zip, county, precinct): `page.getByPlaceholder(...)` or `page.getByLabel(...).fill()`
