  <name>Task 1: Multi-stage Dockerfile and SPA serving in app/main.py</name>
  <files>Dockerfile, app/main.py</files>
  <action>
    **Create `Dockerfile` at project root** with three stages per research recommendations:

    Stage 1 -- `frontend` (FROM node:22-slim):
    - WORKDIR /build
    - COPY web/package.json web/package-lock.json ./
    - RUN npm ci
    - COPY web/ ./
    - RUN npm run build
    - Output at /build/dist/

    Stage 2 -- `deps` (FROM python:3.13-slim):
    - COPY --from=ghcr.io/astral-sh/uv:latest /uv /usr/local/bin/uv
    - WORKDIR /build
    - COPY pyproject.toml uv.lock ./
    - RUN uv sync --frozen --no-dev --no-install-project
    - Output: /build/.venv/

    Stage 3 -- `runtime` (FROM python:3.13-slim):
    - RUN useradd --create-home --shell /bin/bash app
    - WORKDIR /home/app
    - COPY --from=deps /build/.venv ./.venv
    - COPY app/ ./app/
    - COPY main.py ./
    - COPY alembic/ ./alembic/
    - COPY alembic.ini ./
    - COPY --from=frontend /build/dist/ ./static/
    - ARG GIT_SHA=unknown
    - ARG BUILD_TIMESTAMP=unknown
    - ENV GIT_SHA=$GIT_SHA BUILD_TIMESTAMP=$BUILD_TIMESTAMP
    - ENV PATH="/home/app/.venv/bin:$PATH"
    - ENV WORKERS=1
    - USER app
    - EXPOSE 8000
    - CMD ["sh", "-c", "uvicorn main:app --host 0.0.0.0 --port 8000 --workers ${WORKERS:-1} --log-level info"]

    Note on system dependencies: The research flagged that `shapely` may need `libgeos-dev` in the runtime image. Add `RUN apt-get update && apt-get install -y --no-install-recommends libgeos-dev && rm -rf /var/lib/apt/lists/*` in the runtime stage BEFORE the USER directive. If the build succeeds without it and the import test passes, it can be removed in a follow-up, but including it is safer.

    **Update `app/main.py`** to add SPA static file serving:
    - Add imports: `from pathlib import Path`, `from fastapi.staticfiles import StaticFiles`, `from fastapi.responses import FileResponse`, `from starlette.exceptions import HTTPException as StarletteHTTPException`
    - Define `STATIC_DIR = Path(__file__).resolve().parent.parent / "static"` (resolves to /home/app/static/ in container)
    - After existing router includes (health_router, v1_router), add conditional static serving:
      ```python
      if STATIC_DIR.exists():
          app.mount("/assets", StaticFiles(directory=STATIC_DIR / "assets"), name="assets")

          @app.get("/{path:path}")
          async def spa_fallback(path: str):
              if path.startswith("api/") or path.startswith("health/"):
                  raise HTTPException(status_code=404, detail="Not found")
              return FileResponse(STATIC_DIR / "index.html")
      ```
    - The `if STATIC_DIR.exists()` guard ensures the app works in dev mode (no built frontend) without errors
    - The path guard prevents SPA catch-all from swallowing API 404s (per research pitfall 3)
    - Use `from fastapi import HTTPException` for the 404 raise (already imported by fastapi)
