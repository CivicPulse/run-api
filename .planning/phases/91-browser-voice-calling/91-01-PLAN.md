| Boundary | Description |
|----------|-------------|
| Org credential storage | API Key secrets encrypted at rest, never returned in plaintext |
| Campaign RLS | call_records scoped to campaign via RLS policy |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-91-01 | Information Disclosure | Organization.twilio_api_key_secret_encrypted | mitigate | Fernet encryption using existing keyring; never returned in API responses or logs |
| T-91-02 | Elevation of Privilege | call_records table | mitigate | RLS policy on campaign_id matching app.current_campaign_id session var |
| T-91-03 | Tampering | CallRecord.status | mitigate | Status updates only via webhook handler (server-side); frontend never writes call_records directly |
</threat_model>
