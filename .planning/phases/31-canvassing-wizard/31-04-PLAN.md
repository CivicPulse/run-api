2. Create `web/src/components/field/ResumePrompt.tsx`:

Export a hook, not a component:
```typescript
export function useResumePrompt(params: {
  walkListId: string | null
  storedWalkListId: string | null
  currentAddressIndex: number
  totalAddresses: number
  lastActiveAt: number
  onResume: () => void
  onStartOver: () => void
})
```

Logic:
- Detect interrupted session: `walkListId === storedWalkListId && currentAddressIndex > 0 && lastActiveAt > 0`
- On mount (via `useEffect`), if interrupted session detected, show a sonner toast:
  ```typescript
  toast("Pick up where you left off?", {
    description: `Door ${currentAddressIndex + 1} of ${totalAddresses}`,
    action: { label: "Resume", onClick: onResume },
    cancel: { label: "Start Over", onClick: onStartOver },
    duration: 10000,
    onAutoClose: () => onResume(),  // Auto-resume after 10 seconds
  })
  ```
- The `onResume` callback does nothing (stays at current index)
- The `onStartOver` callback: find first PENDING address index and call `jumpToAddress(index)`. This is computed from households where NOT all entries are completed/skipped. Does NOT clear completed entries per CONTEXT.md decision.
- Add countdown text: update toast description with "Resuming in {seconds}..." using a `setInterval` that counts down from 10.

The hook runs once on mount via `useRef` guard to prevent duplicate toasts on re-renders.
