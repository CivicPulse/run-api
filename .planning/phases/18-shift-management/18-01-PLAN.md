   Export these:
   - `shiftKeys` object with: `all(campaignId)`, `detail(campaignId, shiftId)`, `volunteers(campaignId, shiftId)`, `list(campaignId, filters?)` -- following volunteerKeys pattern
   - `useShiftList(campaignId, filters?: { status?: string; type?: string })` -- GET /shifts with query params, returns `PaginatedResponse<Shift>`
   - `useShiftDetail(campaignId, shiftId)` -- GET /shifts/{shiftId}, returns `Shift`
   - `useCreateShift(campaignId)` -- POST /shifts, body: ShiftCreate, invalidates shiftKeys.all
   - `useUpdateShift(campaignId, shiftId)` -- PATCH /shifts/{shiftId}, body: ShiftUpdate, invalidates all + detail
   - `useUpdateShiftStatus(campaignId, shiftId)` -- PATCH /shifts/{shiftId}/status, body: ShiftStatusUpdate, invalidates all + detail
   - `useDeleteShift(campaignId)` -- takes shiftId as mutationFn arg, DELETE /shifts/{shiftId}, invalidates all
   - `useSelfSignup(campaignId, shiftId)` -- POST /shifts/{shiftId}/signup, no body, invalidates all + detail + volunteers
   - `useCancelSignup(campaignId, shiftId)` -- DELETE /shifts/{shiftId}/signup, invalidates all + detail + volunteers
   - `useAssignVolunteer(campaignId, shiftId)` -- takes volunteerId as mutationFn arg, POST /shifts/{shiftId}/assign/{volunteerId}, invalidates all + detail + volunteers
   - `useRemoveVolunteer(campaignId, shiftId)` -- takes volunteerId as mutationFn arg, DELETE /shifts/{shiftId}/volunteers/{volunteerId}, invalidates all + detail + volunteers
   - `useCheckInVolunteer(campaignId, shiftId)` -- takes volunteerId as mutationFn arg, POST /shifts/{shiftId}/check-in/{volunteerId}, invalidates detail + volunteers
   - `useCheckOutVolunteer(campaignId, shiftId)` -- takes volunteerId as mutationFn arg, POST /shifts/{shiftId}/check-out/{volunteerId}, invalidates detail + volunteers
   - `useAdjustHours(campaignId, shiftId)` -- takes { volunteerId, data: HoursAdjustment } as mutationFn arg, PATCH /shifts/{shiftId}/volunteers/{volunteerId}/hours, body: data, invalidates detail + volunteers
   - `useShiftVolunteers(campaignId, shiftId)` -- GET /shifts/{shiftId}/volunteers, returns `PaginatedResponse<ShiftSignupResponse>`, enabled when both IDs present
