  <action>
    **CallListService** (`app/services/call_list.py`):

    1. `generate_call_list(session, campaign_id, data: CallListCreate, user_id)` -> CallList:
       - If data.voter_list_id provided, use build_voter_query with the voter list's filters. Otherwise use a base query for all campaign voters.
       - JOIN VoterPhone to get phone numbers per voter
       - LEFT JOIN do_not_call to exclude DNC numbers (filter where dnc.id IS NULL for each phone)
       - Apply basic phone validation: regex `^\\d{10,15}$` to filter invalid numbers
       - For each voter with at least one valid non-DNC phone: create CallListEntry with phone_numbers JSONB (list of {phone_id, value, type, is_primary} ordered primary-first), priority_score from calculate_priority_score()
       - Create CallList with status=DRAFT, set total_entries count
       - Return the created CallList

    2. `calculate_priority_score(interaction_count: int)` -> int: standalone function. score = 100 - (interaction_count * 20), min 0.

    3. `claim_entries(session, call_list_id, caller_id, batch_size=5)` -> list[CallListEntry]:
       - Load CallList to get claim_timeout_minutes and cooldown_minutes
       - Release stale claims: UPDATE entries WHERE status=IN_PROGRESS AND claimed_at < now - timeout -> set AVAILABLE, clear claimed_by/claimed_at
       - Recycle cooled-down entries: UPDATE entries WHERE status=AVAILABLE AND last_attempt_at IS NOT NULL AND last_attempt_at < now - cooldown AND attempt_count < max_attempts
       - SELECT ... FOR UPDATE SKIP LOCKED: get batch_size entries WHERE status=AVAILABLE ORDER BY priority_score DESC
       - UPDATE claimed entries: status=IN_PROGRESS, claimed_by=caller_id, claimed_at=now
       - Return claimed entries

    4. `update_status(session, call_list_id, new_status)` -> CallList: enforce draft->active->completed only.

    5. `get_call_list(session, call_list_id)`, `list_call_lists(session, campaign_id)`, `delete_call_list(session, call_list_id)` -- standard CRUD.

    **DNCService** (`app/services/dnc.py`):

    1. `add_entry(session, campaign_id, phone_number, reason, added_by)` -> DoNotCallEntry: INSERT with ON CONFLICT DO NOTHING on (campaign_id, phone_number). Return existing if duplicate.

    2. `bulk_import(session, campaign_id, csv_content: str, added_by)` -> DNCImportResponse: Parse CSV with csv.DictReader. Require `phone_number` column. Optional `reason` column (default REGISTRY_IMPORT). Validate phone format (same regex). Batch insert, skip duplicates. Return counts {added, skipped, invalid}.

    3. `delete_entry(session, dnc_id)` -> None

    4. `check_number(session, campaign_id, phone_number)` -> DNCCheckResponse: Query do_not_call for match.

    5. `list_entries(session, campaign_id)` -> list[DoNotCallEntry]: List all DNC entries for campaign.

    Replace test stubs in test_call_lists.py and test_dnc.py with real implementations. Mock AsyncSession. Test the service logic thoroughly.
