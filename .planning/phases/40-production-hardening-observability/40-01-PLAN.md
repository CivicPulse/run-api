2. Create `app/core/middleware/request_logging.py` — pure ASGI middleware (NOT BaseHTTPMiddleware per research):
   - Class `StructlogMiddleware` with `__init__(self, app)` and `async def __call__(self, scope, receive, send)`.
   - Skip non-HTTP scopes: `if scope["type"] != "http": await self.app(scope, receive, send); return`.
   - Skip health checks (per D-14): extract path from `scope["path"]`, if `path.startswith("/health"): await self.app(scope, receive, send); return`.
   - Extract or generate request_id (per D-08): read `X-Request-ID` from `scope["headers"]` (headers are bytes tuples). If missing, generate `uuid.uuid4().hex`.
   - Set ContextVars: `request_id_var.set(request_id)`. Extract `user_id` and `campaign_id` from scope where possible (campaign_id from path pattern `/api/v1/campaigns/{uuid}/`; user_id set later via JWT — leave as empty string initially since JWT parsing happens downstream).
   - Time the request: `start = time.perf_counter()`.
   - Capture status_code via a wrapper around `send`: intercept `http.response.start` message to read `status` field and `content-length` header.
   - After response completes, also extract from scope headers: `user-agent`, `cf-connecting-ip` or `x-real-ip` for client_ip (with fallback to `scope.get("client", ("",))[0]`), query_string from `scope["query_string"]`.
   - Set Sentry tags via `sentry_sdk.set_tag(...)` for request_id, user_id (if available from ContextVar by this point), campaign_id, http_path.
   - Emit structlog JSON log: `structlog.get_logger().info("request", request_id=request_id, user_id=user_id_var.get(""), campaign_id=campaign_id_var.get(""), method=scope["method"], path=path, status_code=status_code, duration_ms=round((time.perf_counter() - start) * 1000, 2), query_params=scope.get("query_string", b"").decode(), response_size=content_length, client_ip=client_ip, user_agent=user_agent)`.
   - Add `X-Request-ID` response header: inject into the `http.response.start` message's headers list.
   - Configure structlog at module level: `structlog.configure(processors=[structlog.stdlib.add_log_level, structlog.processors.TimeStamper(fmt="iso"), structlog.processors.JSONRenderer()], wrapper_class=structlog.stdlib.BoundLogger, context_class=dict, logger_factory=structlog.PrintLoggerFactory())`.
